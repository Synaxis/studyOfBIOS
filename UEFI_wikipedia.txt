UEFI is abreviation for Unified Extensible Firmware Interface.
it defines a software interface between an operating system and platform firmware.
UEFI replaces the legacy Basic Input/Output System(BIOS) firmware interface originally present in all IBM PC personal computers
With most UEFI firmware implementations providing support for legacy BIOS services. UEFI can support remote diagnostics and repair of computers.
even with no operating system installed.
